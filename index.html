Welcome to CrowDyVis a software tool for Real Time Visualization of Crowd Dynamics.

With CrowDyVis you can translates your crowd dynamics data to a 3D real time visualization giving you the possibility to view your crowd dynamics visualization however you want it. CrowDyVis is an application written with the free version of Unity, a game development tool, and is scripted in C#. On this website you will find information on how to use CrowDyVis, information about the project and a link to the github repository. For a demonstration please take a look at our youtube channel.

About us

The organization behind CrowDyVis consists of 4 students in collaboration with 2 supervisors from the University of Technology in Eindhoven (TU/e). This project has been executed in collaboration with the Institute for Complex Molecular Systems (ICMS) also located in Eindhoven. This project is started by the Honors Horizon Program from the TU/e and involved the creation of a self-made crowd dynamics simulation made in MATLAB, an Unity application to visualize crowd dynamics (CrowDyVis) and a report describing the project.

How to use CrowDyVis

In order to use CrowDyVis you will need to provide two XML files. The first XML file contains the pedestrian data (the objects and their respective trajectories) and the second XML file contains the geometry (walls, obstacles and so on). For a more in depth explanation please take a look at the User Manual to be found here.
